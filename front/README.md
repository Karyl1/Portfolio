## Cloner le projet :

	git clone https://github.com/Karyl1/RandoSmart

	cd randosmart/

## Lancer le backend:

	cd back/

	npm install

	npm start

le serveur se lancera sur le port 8080

## Lancer le frontend:

	cd front/

	npm install

	npm start
	
le serveur se lancera sur le port 3000
